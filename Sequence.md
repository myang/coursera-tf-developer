## Time Series
* Type of time series data:
  * weather
  * stock market
  * github commitment
* Matrix
  * trend
  * seasonality
* Prediction
  * last value as label
  * moving average
* RNN
  * typically to use two SimpleRNN layers: the first one with return = True, the other without.
  * Bidirectional(): two or more layers with return = True except the last layer.
* LSTM
  * LSTM()
